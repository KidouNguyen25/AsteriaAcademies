# Asteria Academies

Mobile-first 3D math maze puzzle game for Grade 6 and Grade 7, built with Three.js and TypeScript.

## GitHub Repository

Repository: https://github.com/KidouNguyen25/AsteriaAcademies

## Local Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## GitHub Pages

This project is configured for GitHub Pages deployment through GitHub Actions.
